## ðŸ¤ Social Layer (Viral Growth)

FocusPet is built to be played _together_.

### 1. "Pet Dates" (Social Casting)

- **Cast Action:** Cast your pet to a friend's feed using a Frame.
- **Interaction:** If they interact back, both pets "Play".
- **Benefit:** Boosts **Happiness** for both + Bonus XP.

### 2. Accountability Circles

- Form a **"Pack"** (Group of 3-5 users).
- **Pack Bonus:** If _everyone_ in the pack completes their daily focus, the whole group gets a **1.5x XP Boost**.
- **Peer Pressure:** If one person misses, the bonus is lost for everyone.

### 3. Leaderboards

- **"Deep Work Beast":** Most hours focused this week.
- **"Streak Master":** Longest active daily streak.
